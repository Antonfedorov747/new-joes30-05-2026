# Joe's Repair Landing Page

Static mobile landing page prepared from the Figma design for GitHub Pages.

## Files

- `index.html` - page markup
- `styles.css` - responsive styling
- image files such as `logo.png`, `hero-bg.png`, `repair-card.png`, and icons

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and all `.png` image files into the repository root.
3. Open repository `Settings`.
4. Go to `Pages`.
5. In `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save and wait for GitHub to show the public page URL.

Example URL format:

```text
https://your-username.github.io/repository-name/
```
